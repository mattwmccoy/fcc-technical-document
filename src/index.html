What is HTML?

HTML is the standard markup language for creating web pages and web applications.

HTML is able to make very simple webpages, as well as very complex webpages.

Most websites on the World Wide Web are written using HTML.

HTML Elements

Elements are the building blocks of HTML.

Elements are visible on screen once all the code is written.

Examples of HTML elements are:

  • A text header
  • A paragraph
  • A link
  • A list
  • An image

    • HTML Tags

    Elements must be placed within an opening tag and closing tag in order to properly function.

    For example, this is how you write a header:

    <h1>Hello</h1>

    In this instance, the opening tag is <h1> and the closeing tag is </h1>. Closing tags must always have a slash included in order to function.

    How to write HTML

    Headers are commonly followed by paragraphs or lists, which need to be written.

    An example of how to write a paragraph:

    <p>Your text goes here! You can write as much as you like!</p>

    An example of how to write a list:

    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>

    Adding an image with HTML

    Every good website needs a good image, and through HTML images can be added to your website

    How to add an image

    <img src="http://www.images.com/img.jpeg" alt="your image">

    All img elements must have an alt attribute, which is used for screen readers to improve accessibility and is displayed if the image fails to load.

    How to style HTML

    Websites need some flair and pizzaz in order to look proper.

    Through CSS, we can styilize our HTML to give it a unique display

    Example of how to style HTML:

    <style>
    background-color: blue;
    color: gold;
    </style>

    Using <style> we can apply CSS to our web page.

    This set of code will make the background of your web page blue, and it will make the text gold!
